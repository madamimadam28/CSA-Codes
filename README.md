# CSA Class Codes

This repository contains my Computer Science A (CSA) class code assignments and exercises covering topics such as:

- Primitive Types and Variables
- Objects and Classes
- Boolean Expressions and Control Flow
- Iteration and Loops
- Class Design and Object-Oriented Programming
- Arrays and ArrayLists
- 2D Arrays
- Inheritance and Polymorphism
- Recursion

Each folder is organized by topic with individual Java files for different exercises and assignments.
